# 🎉 Bem-vindo à API de Promoções!

Obrigado por se interessar pelo projeto!

## 🚀 Início Rápido

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/api-promocoes.git
cd api-promocoes

# Instale as dependências
npm install

# Configure o ambiente
cp .env.example .env
# Edite o .env com suas configurações

# Configure o banco de dados
npx prisma generate
npx prisma db push

# Inicie o servidor
npm run dev
```
